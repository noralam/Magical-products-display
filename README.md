## 📖 Overview

**Magical Shop Builder** (formerly *Magical Products Display*) is the most powerful **free WooCommerce Template Builder for Elementor**. With **60+ dedicated widgets** and a visual drag-and-drop interface, you can design every page of your WooCommerce store — from product listings to checkout flows — with complete freedom.

🔗 [Website](https://wpthemespace.com/magical-shop-builder) · [Documentation](https://mp.wpcolors.net/doc/) · [Demo](https://mp.wpcolors.net/) · [Pro Version](https://wpthemespace.com/product/magical-shop-builder/)

- 🏗️ **Complete Template Builder** — Create and assign custom templates for any WooCommerce page (like Elementor Pro Theme Builder, but FREE)

## 📥 Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through **Plugins → Installed Plugins**
3. Make sure **WooCommerce** and **Elementor** are installed and active
4. Go to **Magical Shop Builder** in your WordPress dashboard to configure settings
5. Edit any page with Elementor and find the Magical Shop Builder widgets in the panel
